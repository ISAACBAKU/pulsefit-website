# PulseFit Website

A full-stack fitness coaching website built as a realistic client-style portfolio project.

## Overview

PulseFit is a responsive business website for a fitness coaching brand. It includes a modern front-end, mobile navigation, a backend contact form system, submission storage, and automated email notifications.

## Features

- Responsive landing page
- Mobile hamburger menu
- Smooth animations and section reveals
- Contact form with backend API
- Form submissions saved to JSON
- Email notification sent to site owner
- Confirmation email sent to user
- Environment variables managed with dotenv

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- Nodemailer
- dotenv

## Purpose

This project was built to simulate a real client website for a fitness coaching business, while demonstrating front-end, back-end, and full-stack integration skills.
